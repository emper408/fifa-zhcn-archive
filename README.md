# fifa-zhcn-archive

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式保存，不依赖特定域名或网站，仅供存档和查阅用途。

## 目录结构

```
fifa-zhcn-archive/
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面所需的资源文件（如图片、样式、脚本）
└── README.md       # 本文件
```

各目录说明：

- `pages/`：每个子目录或文件对应一个独立的 HTML 页面归档。
- `assets/`：按需存放页面引用的公共资源，例如图片、CSS、JavaScript 等。

## 页面归档说明

- 所有归档页面均为独立的 HTML 文件，可直接在浏览器中打开。
- 页面内容保持原始状态，未做修改或删减。
- 归档时间、来源等信息可能记录在页面源代码的注释中（如有）。
- 部分页面可能依赖于外部资源（如字体、CDN 库），但已尽量内联或本地化。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在 `pages/` 目录下找到感兴趣的 HTML 文件。
3. 使用浏览器直接打开该文件即可查看。
